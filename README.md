### RuneLite Expression-Evaluator
> A runelite plugin that solves mathematics equations in chat messages.

### Supports:
> + *addition* **[ \+ ]**
> + *subtraction* **[ \- ]**
> + *division* **[ \/ ]**
> + *multiplication* **[ \* ]**
> + *modulus* **[ \% ]**
> + *power*  **[ \^ ]**
